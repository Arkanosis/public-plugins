Mus musculus News

    Ensembl Human and Mouse have received their regular cDNA updates
  • Updates to Mouse databases.
    The Ensembl mouse gene set has been patched to remove some incorrect gene structures and to add a set of Immunoglobulin/T-cell receptor gene segments, inferred from the alignment of segment sequences in the IMGT database..

  • Xref updates
    Xrefs have been re-run for Human, Rat and Mouse.
  • Mouse Variation database
    The mouse variation database has been updated to change sample_ids in the strain table and reflect changes in the patched gene set.
