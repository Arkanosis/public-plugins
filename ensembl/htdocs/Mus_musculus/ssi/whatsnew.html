What's New in Ensembl 50

Mus musculus News

  • Updated karyotypes
    Human and mouse karyotypes have been updated with more accurate data from UCSC and NCBI respectively.
    Read more...
  • Non-coding genes
    These have been updated for most species, including an miRNA update and HGNC names where possible.
  • cDNA update
    Human and mouse cDNAs have been updated as usual.
  • Raw compute updates
    The raw computes that were previously run on contigs have been re-run using toplevel sequences.
  • Vega updates
    New Vega genesets have been released for Human and Mouse; the human set includes the Havana annotations that have been merged into Ensembl Human.
