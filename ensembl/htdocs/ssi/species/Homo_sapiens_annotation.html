Annotation

In release 60 (November 2010), we continue to display a joint gene set based on the merge between the automatic annotation from Ensembl and the manually curated annotation from Havana. This refined gene set corresponds to GENCODE release 5.The Consensus Coding Sequence (CCDS) identifiers have also been mapped to the annotations. More information about the CCDS project is available here.
