What's New in Ensembl 33

Species News

Homo sapiens

  • New Human cDNAs
    Monthly update of the human cDNA database.
  • Vega Human
    Updated Ensembl-Vega database for Homo sapiens.
  • Gene descriptions
    The missing gene descriptions for ncRNAs have been added to the Human core database.

Mus musculus

  • Mouse BAC Clones
    Entries have been added to the assembly so that BAC clones can be fetched.
  • Transcript supporting evidence
    The supporting evidence table IDs have been adjusted so that all transcripts show the correct evidence.

Danio rerio

  • Updates to gene table
    The Zebrafish core and est databases had incorrect values for biotype and confidence in the gene table, resulting in EST genes being displayed in the wrong colour and without exons. This has been corrected in release 33.

Rattus norvegicus

  • Xrefs re-run
    Rat xrefs have been re-run to correct problems with RGD mappings.

Drosophila melanogaster

  • New Xrefs
    Additional xrefs for fly.

Gallus gallus

  • Xrefs re-run
    Chicken xrefs have been re-run to improve assignment of display xrefs, particularly for HOX genes.

Ciona intestinalis

  • New Xrefs
    Ensembl Ciona gene predictions which match the JGI's predictions on assembly 1.0 now have xrefs which link through to the JGI website.
  • Gene descriptions
    The truncated descriptions in the Ciona gene table have been replaced with full descriptions.

Data updates

HUGO name changes

The display name for the HUGO database has been changed to 'HGNC symbol' in order to match current usage by SwissProt, etc.

New variation databases

All species have new variation databases.

Affy mapping

Some AffyMetrix data was accidentally omitted from the last release of Ensembl (v32, July 2005). These data have been re-computed and added to the core databases of the affected species.

Fixes to affy_feature

The affy_feature table in the core database had no values in the 'mismatches' column in the last release; the correct values have been added.

Compara

Addition of paralogues for mouse, drosophila and zebrafish.

Web features

Bug fixes

A number of small bugs in the new-style view pages have been fixed, including missing popup menus in SyntenyView.

API/web code changes

Perl method status

The Perldoc for the Perl API has been expanded to indicate the status of each method: 'stable', 'at risk', or 'medium risk'.

Schema changes

New column in external_db

A new column, db_display_name, has been added to external_db, to replace the hardcoded web code hashmap previously used, and the API updated to support the change.
