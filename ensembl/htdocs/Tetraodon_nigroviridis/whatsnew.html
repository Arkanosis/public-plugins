What's New in Ensembl Tetraodon nigroviridis

Data updates

HUGO name changes

The display name for the HUGO database has been changed to 'HGNC symbol' in order to match current usage by SwissProt, etc.

New variation databases

All species have new variation databases.

Web features

Bug fixes

A number of small bugs in the new-style view pages have been fixed, including missing popup menus in SyntenyView.

API/web code changes

Perl method status

The Perldoc for the Perl API has been expanded to indicate the status of each method: 'stable', 'at risk', or 'medium risk'.

Schema changes

New column in external_db

A new column, db_display_name, has been added to external_db, to replace the hardcoded web code hashmap previously used, and the API updated to support the change.
