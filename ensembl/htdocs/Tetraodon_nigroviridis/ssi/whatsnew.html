What's New in Ensembl 50

Tetraodon nigroviridis News

  • New Tetraodon Build
    A new Tetraodon database has been released in Ensembl 50. The core database contains a standard Ensembl build, whilst the otherfeatures database contains the data that was previously released on Ensembl Pre.
  • New variation databases
    Ensembl 50 includes a new orangutan variation database, with SNPs called by Ensembl using ssaha2/pileup from reference assembly reads, plus remapping of SNPs to the new assemblies for tetraodon and cow.
    Read more...
  • Non-coding genes
    These have been updated for most species, including an miRNA update and HGNC names where possible.

General News

  • Canonical Transcripts

    Canonical transcripts have been defined for all genes in the core databases.
    Read more...

  • SSAHA
    From release 50 we will no longer be providing SSAHA sequence search. If you wish to run your own SSAHA sequence search you can download the files to generate the search hashes from our FTP site.
