What's New in Ensembl 49

Drosophila melanogaster News

  • New import of Fly database
    Ensembl Fruitfly now includes the FlyBase 5.4 annotations. See the Fruitfly home page for more details.
  • Functional Genomics
    eFG now includes a database for fruitfly to hold regulatory features formerly stored in the core database, e.g. BioTIFFIN/flyreg.
