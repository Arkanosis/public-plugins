Accessing Ensembl Data

Export data from the website

Many of the pages displaying Ensembl genomic data offer an export option, suitable for small amounts of data, e.g. a single gene sequence.

Look for the 'Export' button in the left-hand menu.

Create a custom download

Complex datasets can be retrieved using the BioMart data-mining tool.

You may find exploring this web-based data mining tool easier than extracting information from our normalised database dumps.

Fetch data with our Perl API

For rapid access to small-to-medium datasets, we recommend use of our Perl API.

The API allows you to quickly write scripts to retrieve and query biological data objects such as genes, variation features and 'slices' of DNA sequence.

# Fetch all clones from a slice adaptor 
# (returns a list reference)
my $clones_ref = $slice_adaptor->fetch_all('clone');

The API can connect to our publicly-accessible MySQL server, so you may not need to download complete databases.

Large datasets (file dumps via FTP)

If required, entire databases can be downloaded from our FTP site in a variety of formats, from flat files to MySQL dumps.

Please be aware, however, that an entire EnsEMBL release currently totals almost a terabyte of MySQL files, hence the other tools mentioned above may be more suitable for small queries!

Web code

If your project makes heavy use of Ensembl data or wishes to display confidential data alongside Ensembl tracks, you can download our webcode and set up your own personal mirror site.

The web code is designed to be highly customisable, so you can brand your site, install only a subset of species, or even create a completely new site based on our web interface and/or drawing code.
