Ensembl Roadmap

The following updates are planned for upcoming releases of Ensembl.

Please note that we have no fixed timeline for most of these items

Gene annotation

  • Builds in progress: Human, Western clawed frog (Xenopus tropicalis), White-cheeked gibbon (Nomascus leucogenys)
  • New species expected: Sea lamprey (Petromyzon marinus), Atlantic cod (Gadus morhua), Cynomolgus macaque (Macaca fascicularis), Turquoise killifish (Nothobranchius furzeri), Tasmanian devil (Sarcophilus harrisii)
  • Updated assemblies expected for: mouse, rat, pig, microbat, bushbaby and gorilla
  • Incorporation of new non-reference sequence for human, with basic gene annotation
  • Incorporation of HAVANA manual annotation into the human gene set for every release, mouse gene set for every second release, and zebrafish gene set at least once a year
  • Tissue specific transcript models based on RNASeq data for human, based on the Illumina Human Body Map 2.0 data

Comparative Genomics

  • Bird multiple alignments
  • Extend the EPO multiple alignment pipeline to all vertebrates
  • Display super-protein tree
  • HMM-based classification of proteins for Families and Gene Trees
  • Annotate ohnologs
  • Display gene loss and gain events

Variation updates

  • continue to import new variation data from dbSNP and DGVa where available
  • improve variation annotation using data from the 1000 Genomes Project
  • continue to import genome wide association study phenotypes for variants from the NHGRI catalog
  • database schema changes to associate variation consequence is a specific variant alleles rather than just a locus
  • incorporate sequence ontology terms in order to add an ontology to the variation consequences
  • extend variation consequence pipeline to consider specific allele changes within regulatory regions
  • extend variation consequence pipeline to include predictions using SIFT and/or PolyPHEN

Core API and schema

  • Support for alternative initiation
  • Support for operons
  • Refinements to stable ID mapping
  • Improved support for assembly exceptions (including haplotypes and "patch" sequences)
  • Support for alternative alleles
  • Improved Perl API documentation
  • Refinements to gene name and description assignment

Functional Genomics updates

  • Incorporating DNase1 footprinting and sequence conservation in to TFBS identification
  • Integrating RNA-seq data with regulatory element annotation
  • Developing meta-annotation of Multi-cell regulatory elements.
  • Development of data tracking database.
  • Fine tuning of peak-calling parameters for histone modification ChIP-seq, including investigation of alternate methods.
  • Further development of uniform signal processing.
  • Removal of collections data from the functional genomics database.
  • eFG Tools:
    • Nearest gene
    • PWM consequences
  • Web display developments:
    • Further refinements of wiggle track config/display including track highlighting.
    • Data source display, including links to outside data sources for ChIP-seq data, antibodies, PWMs, etc.
  • Incorporating ChIP-seq data from further species for possible additional regulatory builds e.g. Schmidt et al (PMID: 20378774).

New web features

  • 'Export image' buttons on karyotypes and other 'vertical' images
  • Ability to save user configurations
  • Ability to create a custom page from a selection of existing components

Biomart

  • Test new interface
  • Test new configurator tool
  • Migrate to new building code and tools if testing is successful
