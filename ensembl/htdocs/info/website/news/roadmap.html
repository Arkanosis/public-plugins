Ensembl Roadmap

The following updates are planned for upcoming releases of Ensembl.

Please note that we have no fixed timeline for most of these items

Gene annotation

  • Builds in progress: human, mouse, zebrafish, panda
  • New species expected: lamprey, gibbon, cod, tasmanian devil, buffalo
  • Updated assemblies expected for mouse, rat, bat, anolis lizard, fugu, gorilla, zebrafinch, chimpanzee
  • Incorporation of RNA-Seq-based models into the zebrafish gene set
  • Tissue-specific transcript models in human, based on RNA-Seq data (including data from Illumina's Human Body Map 2.0 project)

Comparative Genomics

  • Bird multiple alignments
  • HMM-based classification of proteins for Families and Gene Trees

Variation updates

  • Import of malarial dataset from the EGA (European Genome-phenome Archive)
  • Update list of variants with UniProt annotation
  • New import of variants with phenotype annotation from NHGRI
  • New import of dbSNP 132 databases when available, including data from the 1000 Genomes project
  • Improve the web displays based on LRG sequences

Core API and schema

  • Support for operons

Biomart

  • Test new interface
  • Test new configurator tool
  • Migrate to new building code and tools if testing is successful

eFG updates

  • Improve documentation of eFG in the user interface, including updated API tutorial, and describe developments through the blog.

New web features

  • 'Export image' buttons on karyotypes and other 'vertical' images
  • Replacement for GOView, using our new GO schema and API
  • Ability to save user configurations
  • Ability to create a custom page from a selection of existing components
  • "Ensembl Tools"
    • Make data-munging "wizards" (Assembly Converter, etc) available as downloadable scripts and via a BLAST-like ticketted job submission system
    • Complete rewrite of BLAST interface and backend code

Note that many of the above features require major work on the web infrastructure, so will take some time to achieve.
