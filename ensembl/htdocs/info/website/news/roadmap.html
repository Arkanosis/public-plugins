Ensembl Roadmap

The following updates are planned for upcoming releases of Ensembl.

Please note that we have no fixed timeline for most of these items

Gene annotation

  • Builds in progress: duck, rabbit
  • New species expected: lamprey, gibbon, cod
  • Updated assemblies expected for mouse, rat, marmoset, bat and gorilla
  • Refinement of LincRNA prediction pipeline and extension to mouse
  • Incorporation of RNASeq based models in zebrafish gene set
  • Tissue specific transcript models based on RNASeq data
  • Improvements to selenocysteine containing gene predictions

Comparative Genomics

  • Fish multiple alignments
  • Bird multiple alignments
  • Homology predictions for ncRNAs
  • HMM-based classification of proteins for Families and Gene Trees

Variation updates

  • import of malarial dataset from the EGA (European Genome-phenome Archive)
  • update list of variants with UniProt annotation
  • new import of variants with phenotype annotation from NHGRI
  • new import of dbSNP 131 databases when available, including data from the 1000 Genomes project
  • support for indels on the resequencing website page
  • web displays based on LRG sequences
  • karyotype display with all variant positions associated with a particular phenotype

Core API and schema

  • Support for alternative initiation
  • Support for operons
  • Refinements to stable ID mapping
  • Support for LRG (Locus Reference Genomic) sequences
  • Improved support for assembly exceptions (including haplotypes and "patch" sequences)
  • Refinements to gene name and description assignment

Biomart

  • Test new interface
  • Test new configurator tool
  • Migrate to new building code and tools if testing is successful

eFG updates

  • Incorporation of CD133+ and CD36+ Haematopoietic Stem Cell histone modification data into Ensembl under the current schema (Cui et al 2009).
  • Incorporation of the first ENCODE histone modification, TFBS and DNase1 data into Ensembl under the current schema.
  • Incorporation of at least one set of Epigenomics Roadmap project data of histone modification data for human ES cells.
  • Improve documentation of eFG in the user interface, including updated API tutorial, and describe developments through the blog.
  • Perform cell-specific regulatory builds
  • Make (some) data visible in the region in detail view of the interface as wiggle type tracks.
  • Development of the regulatory feature view

    • New web features

    • 'Export image' buttons on karyotypes and other 'vertical' images
    • Replacement for GOView, using our new GO schema and API
    • Ability to save user configurations
    • Ability to create a custom page from a selection of existing components
