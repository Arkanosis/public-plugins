Using Ensembl

There are several ways that you can use Ensembl from your own computer:

  • Browse the provided data on the website
  • Add your own data to our website
  • Retrieve small datasets from our public database server using the Perl API

If you find working over the Internet to be slow, you may find it beneficial to create a local mirror of Ensembl. This will require a fairly powerful server, but full instructions are available to help you, and you can also contact us if you have any problems.
