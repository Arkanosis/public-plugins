Mouse vs Ferret LastZ alignments

Mouse (Mus musculus, GRCm38) and Ferret (Mustela putorius furo, MusPutFur1.0) were aligned using the LastZ alignment algorithm (LastZ) in Ensembl release 69. Mouse was used as the reference species. After running LastZ, the raw LastZ alignment blocks are chained according to their location in both genomes. During the final netting process, the best sub-chain is chosen in each region on the reference species.
