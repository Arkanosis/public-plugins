Anole lizard vs Chinese softshell turtle LastZ alignments

Anole lizard (Anolis carolinensis, AnoCar2.0) and Chinese softshell turtle (Pelodiscus sinensis, PelSin_1.0) were aligned using the LastZ alignment algorithm (LastZ) in Ensembl release 68. Anole lizard was used as the reference species. After running LastZ, the raw LastZ alignment blocks are chained according to their location in both genomes. During the final netting process, the best sub-chain is chosen in each region on the reference species.
