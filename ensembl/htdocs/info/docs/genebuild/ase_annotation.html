Alternative Splicing Event Set

Introduction

The set of alternative splicing events (AS events) is derived from Ensembl gene annotation. The Ensembl gene set includes both automatic and manual annotation, with all transcripts based on experimental evidence. Alternatively spliced transcripts of the same Ensembl gene overlap in coding sequence. These data can be found on the website in the gene and/or transcript tab, and can be accessed by BioMart, or programmatically from the core database using the Perl API.

AS events are available for the following model organisms:

  • Caenorhabditis elegans
  • Danio rerio
  • Drosophila melanogaster
  • Homo sapiens
  • Mus musculus
  • Rattus norvegicus

Common Alternative Spliced Patterns

Seven common AS patterns are automatically annotated in Ensembl for the species mentioned:

AS Pattern TypeAcronymDefinition
Cassette exon (skipped exon)CEOne exon is spliced out of the primary transcript together with its flanking introns.
Intron retentionIRA sequence is spliced out as an intron or remains in the mature mRNA transcript.
Mutually exclusive exonsMXEIn the simpliest case, one or two consecutive exons are retained but not both.
Alternative 3' sitesA3SSAlso called alternatively acceptor sites. Two or more splice sites are recognized at the 5' end of an exon. An alternative 3' splice junction (acceptor site) is used, changing the 5' boundary of the downstream exon.
Alternative 5' sitesA5SSAlso called alternative donor sites. Two or more splice sites are recognized at the 3' end of an exon. An alternative 5' splice junction (donor site) is used, changing the 3' boundary of the upstream exon.
Alternative first exonAFEA pattern of mutually exclusive exon whereby second exons of each variant have identical boundaries but the first exons are mutually exclusive. This is a to allow annotation of possible alternative promoter usage*.
Alternative last exonALEPenultimate exons of each splice variant have identical boundaries, but the last exons are mutually exclusive. This is a to allow annotation of possible alternative polyadenylation usage*.

* AFE and ALE event types are difficult to interpret and biological evidence is needed to support these predictions.

Ensembl imports two additional types of AS events from the ASTD project ([1]).

AS Pattern TypeAcronymDefinition
Intron isoformIIAlternative donor/acceptor splice sites leads to truncation/extension of introns. This is for introns with truncation and/or extension on both splice sites.
Exon isoformEIAlternative donor/acceptor splice sites leads to truncation/extension of exons. This is for exons with truncation and/or extension on both splice sites. First and last exons are never in this computation. Flanking exons must overlap.

Different acronyms for the same type of AS event can be found in the scientific literature and in several online related resources. Ensembl uses CE, IR, MXE, A3SS, A5SS, AFE, ALE, II, EI as standard acronyms. Similar acronyms can be found in [1] and [2].
