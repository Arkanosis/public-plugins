Alternative Splicing Event Set

Introduction

The set of alternative splicing events (or AS events for short) is derived from the set of gene annotation provided by Ensembl. Ensembl gene set includes both automatic annotation and manual curation. Ensembl transcripts are products of the Ensembl automatic pipeline and based on experimental evidence. Alternatively spliced transcripts of the same Ensembl gene are compared to provide a comprehensive list of common alternative splicing events in Ensembl. These data can be retrieved using BioMart or programmatically using the Perl core API.

AS events are available for the following model organisms:

  • Caenorhabditis elegans
  • Danio rerio
  • Drosophila melanogaster
  • Homo sapiens
  • Mus musculus
  • Rattus norvegicus

Common Alternative Spliced Patterns

Seven common AS patterns are automatically annotated in Ensembl for the species aforementioned:

Alternative last exon
AS PatternTypeAcronymDefinition
Cassette exon (skipped exon)CEOne exon is spliced out of the primary transcript together with its flanking introns.
Intron retentionIRA sequence is spliced out as an intron or remains in the mature mRNA transcript.
Mutually exclusive exonsMXEIn the simpliest case, one or two consecutive exons are retained but not both.
Alternative 3' sitesA3SSAlso called alternatively acceptor sites. Two or more splice sites are recognized at the 5' end of an exon. An alternative 3' splice junction (acceptor site) is used, changing the 5' boundary of the downstream exon.
Alternative 5' sitesA5SSAlso called alternative donor sites. Two or more splice sites are recognized at the 3' end of an exon. An alternative 5' splice junction (donor site) is used, changing the 3' boundary of the upstream exon.
Alternative first exonAFEA pattern of mutually exclusive exon whereby second exons of each variant have identical boundaries but the first exons are mutually exclusive. This is to annotate possible alternative promoter usage*.
ALEA pattern of mutually exclusive exon whereby penultimate exons of each variant have identical boundaries but the last exons are mutually exclusive. This is a to annotate possible alternative polyadenylation usage*.

* While AFE and ALE event types are computationally valid, they are difficult to interpret and biological evidences are needed to support alternative promoter/polyadenylation usage.

Ensembl adds two general types of AS events inherited from the ASTD project ([1]) for events not annotated by the standard patterns.

AS PatternTypeAcronymDefinition
Intron isoformIIUse of alternative donor or acceptor splice sites leads to truncation/extension of introns. This is for intron with truncation/extension on both splice sites.
Exon isoformEIUse of alternative donor or acceptor splice sites leads to truncation/extension of exons. This is for exon with truncation/extension on both splice sites. First and last exon are never considered for this computation. Besides, the flanking exons must overlap.

Different acronyms for the same type of AS event can be found in the scientific literature and in several alternative splicing related resources online. Ensembl uses CE, IR, MXE, A3SS, A5SS, AFE, ALE, II, EI as standard acronyms. Similar acronyms can be found in [1] and [2].
