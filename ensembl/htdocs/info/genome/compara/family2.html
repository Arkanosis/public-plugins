Gene Families in Compara

Ensembl families are determined through clustering of all Ensembl and Ensembl Genomes proteins along with metazoan sequences from UniProtKB. It therefore provides a way of exploring orthologues and closely related homologues across a range of animal species. Ensembl Protein Family Report

The pipeline consists of the following steps:

  1. Load members from Ensembl
  2. Load UniProt entries
  3. Run HMM search (using the HMM library described in this document)
  4. Store the families
  5. Align the families with Mafft
  6. Annotate the family with a consensus description (from its members' descriptions)
