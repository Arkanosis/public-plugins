Ensembl Memcached integration

Memcached is a distributed memory object caching system.
We are using memcached to speed up the website by caching data in EnsEMBL, such as semi-static rendered html pages, generated images, image-maps, some database query results and a lot more.
To learn more about memcached in general, visit: http://memcached.org/
