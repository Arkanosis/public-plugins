Ensembl Tools

We provide a number of ready-made tools for processing both our data (Biomart and the Virtual machine) and yours. For the latter:

  • The VEP processes jobs via a ticketing system and can handle medium to large files (up to 50Mb).
  • For other jobs, only small datasets of up to 5Mb can be uploaded and processed online. We aim to extend the ticketing system used for VEP to these tools.
  • For larger datasets we provide an API script that can be downloaded (you will also need to install our Perl API to use these).
